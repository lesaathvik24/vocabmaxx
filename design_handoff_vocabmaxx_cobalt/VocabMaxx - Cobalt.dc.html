VocabMaxx · Cobalt direction

The full redesign, top to bottom

Direction 1a — Cobalt applied across every screen, web and mobile. Same flow you have today; new surface, depth, type and rhythm. Scroll down through Landing → Dashboard → Review → Capture → Words → Insights. Pan / zoom freely.

What I fixed cosmetically
  • Flat teal → cobalt with depth. Soft layered shadows, one glass top bar, gradient only on the hero moment — not everywhere.
  • Type hierarchy. SF-system display, tabular figures for all stats, tighter tracking on headings.
  • Calmer neutrals. Warmer off-white canvas so white cards actually read as elevated.
  • Consistent radii & spacing scale. 16 / 20 / 26px families instead of ad-hoc values.
UX issues I caught
  • No clear "next action." Dashboard now leads with one Due banner as the primary CTA; everything else is secondary.
  • Grade buttons lacked meaning. Again/Hard/Good/Easy now carry color + interval hints so the choice is obvious.
  • Capture had no feedback. Live definition preview appears before you commit, so adding a word feels instant.
  • Words list was a flat table. Added status pills + memory-strength bars so scanning is meaningful.
